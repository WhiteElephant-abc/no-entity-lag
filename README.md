## 主要功能

- 在非玩家实体数大于 1k 时清除所有非玩家实体
- 令所有实体在附近 10 格内实体数大于 50 时清除这些实体，不会杀死玩家以及村民
- 在玩家动作栏（快捷栏上方）显示实时实体数量
- TNT 附近五格内的 TNT 实体数大于 200 时清除这些 TNT 实体
- 每 180s 显示一次实体数量信息
- 每 600s 清理一次掉落物
